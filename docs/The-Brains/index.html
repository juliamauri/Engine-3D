Meet the team behind RedEye!

We are the 2 sole programmers working side by side on RedEye . We started building a simple 3d model viewer for a university assignment and then kept adding new features we found interesting. As a personal challenge and a way to prove to ourselves that we were able to implement these systems, we upgraded rendering with deferred lighting, enhanced resource management and even applied data-oriented design paradigms to improve performance. For our Final Degree Projects, we decided to use the opportunity to push forward the engine with a particle pipeline deployed in the latest release.

Rubén Sardón (left)
